태양계 관측 시뮬레이터

실시간 3D로 구현된 태양계를 자유롭게 비행하며 관측하는 시뮬레이터입니다.
태양과 8개의 행성을 가까이서 살펴보고, 시간 배속과 거리·크기 비율을 바꿔가며 탐험해보세요.

조작 방법

  • W A S D전진 / 후진 / 좌우 이동
  • 마우스 드래그시점 회전 (상하 포함)
  • 마우스 휠확대 / 축소
  • 클릭천체 선택 — 선택하면 카메라가 자동으로 따라가며, 드래그하면 추적이 풀립니다
  • + / -공전 속도 배율 조절
  • Space일시정지 / 재생
  • R실제 비율 ↔ 압축 비율 전환
W A S D : 전진 / 후진 / 좌우 이동  |  마우스 드래그 : 시점 회전 (상하 포함)
마우스 휠 : 확대 / 축소  |  클릭 : 천체 선택 (자동 추적 시작, 드래그하면 해제)
+ / - : 공전 속도 배율 (현재 x1)  |  Space : 일시정지
R : 비율 전환 (현재 압축 비율)
